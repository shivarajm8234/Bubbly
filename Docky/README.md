# ⚔️ Docky

**Docky** is a central control panel for the Bubbly application suite.

### Usage
Run `./start.sh` to launch the dock.

### Features
- Monitor running status of Bubbly apps.
- Start/Stop/Restart with one click or context menu.
- Open configuration folders for each app.
- Sleek, modern PyQt6 interface.
